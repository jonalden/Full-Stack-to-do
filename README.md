# Super-to-do

Super ToDo is a full-stack to do list that utilizes a front-end, a server, and a database to store, update, and delete a users daily tasks. Give it a shot down below.

### https://super-to-do.herokuapp.com

## Technologies used

    * JavaScript
    * Node.js
    * Express
    * MySQL
    * Handlebars
    * Heroku
    * Sequelize
